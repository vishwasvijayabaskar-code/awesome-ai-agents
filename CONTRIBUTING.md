# Contributing to Awesome AI Agents

Thank you for your interest in contributing!

## Guidelines

### Adding a New Project

1. Fork this repository
2. Add your project in the appropriate category
3. Follow the table format:

```markdown
| [Project Name](https://github.com/owner/repo) | Xk+ | Language | Short description. | ✅ Active |
```

4. Submit a pull request

### Requirements

- **Actively maintained** — must have a commit within the last 3 months
- **Open source** — must have a public repository with a license
- **Relevant** — must be related to AI agents, not general AI/ML tools
- **No duplicates** — check existing entries first
- **Star count** — include approximate GitHub stars (e.g., "5k+")

### Maintenance Status

Use these indicators:

| Status | Meaning |
|--------|---------|
| ✅ Active | Regular commits, responsive maintainers |
| ⚠️ Slowing | Less frequent updates, but still alive |
| ⚠️ Pivoting | Major direction change underway |
| ⚠️ Archived | No longer actively developed |

### What NOT to Add

- Closed-source tools with no free tier
- Projects with fewer than 100 GitHub stars
- Tutorials or blog posts (use Learning Resources section)
- Duplicate entries or forks of existing entries

## Removing Dead Projects

If you notice a project that hasn't been updated in 6+ months, please open an issue or PR to remove it.

## Code of Conduct

Be respectful and constructive. We're all here to build awesome things.
